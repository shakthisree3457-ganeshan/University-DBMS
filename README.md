Hospital Management System

Overview

This is a web-based Hospital Management System developed using Python and MySQL. The project is designed to manage patient records, doctor details, appointments, and billing information efficiently. It provides secure data storage and retrieval while reducing manual work and improving hospital management.

Features

- Patient record management
- Doctor information management
- Appointment scheduling
- Billing management
- Add, update, delete, and search records
- Secure database storage

Technologies Used

- Python
- MySQL
- SQL
- HTML
- CSS
- JavaScript

DBMS Concepts Used

- Relational Database Management System (RDBMS)
- Database Design
- Tables and Relationships
- Primary Keys and Foreign Keys
- CRUD Operations (Create, Read, Update, Delete)
- SQL Queries
- Data Integrity and Constraints
- Normalization
- MySQL Database Connectivity using Python

Database

The project uses MySQL as the backend database, connected with Python to perform efficient and reliable database operations.

Future Enhancements

- User authentication and role-based access
- Online appointment booking
- Medical history tracking
- Report generation
- Email and SMS notifications

Conclusion

This project demonstrates the practical implementation of DBMS concepts using MySQL and Python to build a reliable and efficient Hospital Management System.
