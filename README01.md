University-DBMS

Overview

University-DBMS is a database management system developed using Python and MySQL to efficiently manage university-related data. The system stores and manages information such as students, faculty, courses, departments, and enrollments while ensuring organized and secure data handling.

Features

- Student management
- Faculty management
- Course management
- Department management
- Enrollment management
- Add, update, delete, and search records

Technologies Used

- Python
- MySQL
- SQL

DBMS Concepts Used

- Relational Database Management System (RDBMS)
- Entity-Relationship (ER) Model
- Tables and Relationships
- Primary Keys and Foreign Keys
- Normalization
- SQL Queries
- CRUD Operations (Create, Read, Update, Delete)
- Constraints
- MySQL Database Connectivity using Python

Conclusion

This project demonstrates the implementation of core DBMS concepts using Python and MySQL to provide an efficient and reliable university database management system.
